üöÄ Step-by-Step Implementation

1

Setup Project

mkdir user-management-auth cd user-management-auth go mod init user-management-auth go get -u github.com/gin-gonic/gin go get -u gorm.io/gorm go get -u gorm.io/driver/mysql go get github.com/golang-jwt/jwt/v5 go get golang.org/x/crypto/bcrypt go get github.com/joho/godotenv
2

Environment Variables

Create .env file:

DB_HOST=localhost DB_USER=root DB_PASSWORD=yourpassword DB_NAME=user_management DB_PORT=3306 JWT_SECRET=your-super-secret-key-change-this JWT_EXPIRATION=24
Tip: Never commit .env to Git! Add it to .gitignore
3

Database Setup

Create MySQL database:

CREATE DATABASE user_management;
